Templates expressions are not python, if you want to run python code then you add it as a "plugin" using filters or tags (explained in other video in this same django playlist) This has advantages, such as enforcing separation between the ui code and the business logic code, you don't want to mess your html files with a hardly maintanable mixin of python and html code, instead you want to keep it clean, well, django enforces that.
